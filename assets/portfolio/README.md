# Portfolio Images

Place your portfolio project images in this folder.

## Image Requirements

- **Format**: JPG or PNG
- **Dimensions**: 1200 x 800px (or maintain 3:2 aspect ratio)
- **File Size**: Keep under 500KB for optimal loading speed
- **Naming**: Use descriptive names (e.g., `project-1.jpg`, `ecommerce-site.jpg`)

## Adding More Projects

To add more portfolio items, edit `portfolio.html`:

1. Copy an existing `.portfolio-item` div
2. Update the image path in `<img src="assets/portfolio/YOUR-IMAGE.jpg">`
3. Change the `data-category` to either "websites" or "ui-designs"
4. Update the project title, description, and tech tags
5. Customize the overlay content and placeholder icon

The page automatically handles filtering and responsive layout!
